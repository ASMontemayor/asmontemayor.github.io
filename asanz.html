Proyectos Fin de Carrera (PFC`s) en desarrollo (curso 2003/04):
  • Procesamiento de imágenes con lenguajes de alto nivel para la programación de shaders
    Breve descripción: El proyecto consiste en realizar un estudio de las nuevas propuestas de las principales marcas comerciales de hardware gráfico para la programación a bajo nivel de sus tarjetas. En concreto se estudiarán los lenguajes de alto nivel Cg de Nvidia, HLSL de DirectX y Glslang de OpenGL 2.0. Como aplicación se crearán pixel shaders para aplicarlos al procesamiento de altas prestaciones de imágenes.
    Número de participantes: 1 Ing. Sup. Inf.
    Requisitos: C, disposición a aprender DirectX, OpenGL, Cg, Glslang y HLSL
    Adjudicado

  • Desentrelazado de vídeo de altas prestaciones
    Breve descripción: El proyecto consiste en el desarrollo de uno o varios módulos que funcionen bajo un entorno de desentrelazado de libre distribución (Dscaler, VirtualDub o similar). Opcionalmente el objetivo puede diverger a uno más ambicioso para realizar una aplicación de procesamiento de vídeo bajo VRM-9 (Video-Mixing-Rendering-9). Este tipo de aplicaciones tienen especial relevancia para la visualización de vídeo o TV a través del ordenador (Home Theater PC, HTPC).
    Número de participantes: 1
    Requisitos: C, disposición a aprender DirectX
    Adjudicado

  • Ampliación del Optófono
    Breve descripción: El proyecto consiste en ampliar el desarrollo del optófono creado en un trabajo anterior ("Sonificación de imágenes 2-D") introduciendo nuevos métodos de preprocesamiento y segmentación (sonido estéreo, mapas de profundidad de bordes,...)
    Número de participantes: 1
    Requisitos: MATLAB y/o C/C++ y DirectX o disposición a aprender
    Adjudicado

  • Visual-Ratón (co-tutor: Juanjo Pantrigo)
    Breve descripción: El proyecto consiste en el desarrollo de una aplicación que simule el comportamiento de un ratón estándar mediante el seguimiento de alguna característica visual humana identificable (mano, cara, ojos,...). Este tipo de aplicaciones tienen especial relevancia en personas con discapacidad.
    Número de participantes: 1
    Requisitos: C, MATLAB o disposición a aprender
    Adjudicado
